Travel Agency

Project Information

  • Project category: Web App
  • Creation date: November 2025
  • Project URL: github.com/alexoiik/travel-agency

A modern, AI-enhanced dashboard built for travel agencies to manage destinations, bookings, customers, analytics, and AI-powered assistance. Developed using React, React Router v7, Syncfusion UI, Appwrite backend, and Gemini AI.

Tech Stack:

Category Technology
Frontend React, React Router v7
UI Components Syncfusion
Backend Appwrite (Database, Auth, Storage, Functions)
AI Google Gemini API
